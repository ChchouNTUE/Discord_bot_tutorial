# Discord_bot_tutorial
## 4/12 國北教大 Discord Bot 教學

### 下載清單 

- Miniconda--https://docs.conda.io/en/latest/miniconda.html

- Vscode--https://code.visualstudio.com/

- requirement.txt

### Anaconda Prompt 指令
- conda create -n "環境名字" python=3.9     
- conda activate "環境名字"

### Pip 指令
- pip install -r requirement.txt
